M-AIDA Core System v7.0 🚀

Automated Statistical Extraction Engine for Meta-Analysis Research

📑 Project Overview

M-AIDA (Meta-Analysis Intelligent Data Assistant) is an advanced AI-driven framework designed to automate the extraction of statistical parameters (N, r, t, β, df) from unstructured academic PDF papers.

This project was developed at the School of Economics, Can Tho University (CTU) to assist quantitative researchers in optimizing the systematic review and meta-analysis workflow.

🏗️ Technical Architecture

The system utilizes a modern Hybrid Full-stack architecture to ensure high performance and scientific precision:

Frontend Layer (Node.js & React 18): Provides a high-performance academic dashboard for data management and verification.

AI Engine Layer (Python & FastAPI): Implements LLM-based context analysis for accurate statistical extraction.

Database Integration (Notion API): Seamlessly synchronizes verified results into a centralized knowledge workspace.

🛡️ Scientific Integrity

The system features the "Institutional Integrity Secured" workflow:

AI performs preliminary parameter extraction.

The Primary Investigator (PI) performs human-in-the-loop verification.

Data is finalized and locked only after authorized manual confirmation, ensuring absolute research reproducibility.
