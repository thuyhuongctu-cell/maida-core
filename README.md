M-AIDA Core System v7.0 🚀
Automated Statistical Extraction Engine for Meta-Analysis research.
📑 Dự án Xác lập Bản quyền (Project IP Declaration)
M-AIDA (Meta-Analysis Intelligent Data Assistant) là một hệ thống Trí tuệ nhân tạo chuyên sâu được thiết kế để tự động hóa quy trình khai thác dữ liệu thống kê từ các báo cáo khoa học PDF.
Dự án được thực hiện bởi nhóm nghiên cứu tại Trường Kinh tế, Đại học Cần Thơ (CTU), hướng tới tiêu chuẩn liêm chính khoa học và tối ưu hóa hiệu suất nghiên cứu Meta-Analysis.
🏗️ Kiến trúc Hệ thống (Technical Architecture)
Hệ thống áp dụng mô hình Hybrid Full-stack đa nền tảng:
 * Frontend Layer (Node.js & React 18): Dashboard điều phối và giao tiếp người dùng.
 * AI Logic Layer (Python & FastAPI): Bộ não xử lý ngôn ngữ tự nhiên, bóc tách tham số N, r, t, \beta.
 * Database Layer (Notion API): Đồng bộ hóa kết quả vào hệ thống quản trị tri thức nghiên cứu.

🛡️ Liêm chính Khoa học (Scientific Integrity)
